# MqttPipe

This gem wraps the MQTT gem and adds a serializer for simple data structures.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mqtt_pipe'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mqtt_pipe
